# 智撰云 - 房地产酒店营销文案AI助手

## 项目介绍
智撰云是一款专业的AI文案生成工具，专为房地产和酒店行业设计，帮助营销人员快速生成高质量的朋友圈文案。

## 如何部署到GitHub Pages

### 第一步：准备工作
1. 你需要先拥有一个GitHub账号
   - 如果没有账号，请访问 [GitHub官网](https://github.com) 免费注册一个
   - 注册时只需要邮箱、用户名和密码，不需要英语很好

### 第二步：创建新仓库
1. 登录你的GitHub账号
2. 点击右上角的 "+" 图标，选择 "New repository"
3. 在仓库名称中填写：`你的用户名.github.io`（请将"你的用户名"替换为你实际的GitHub用户名）
4. 在描述中可以填写："智撰云 - 房地产酒店营销文案AI助手"
5. 选择 "Public" 公开仓库
6. 勾选 "Add a README file" 选项
7. 点击 "Create repository" 创建仓库

### 第三步：下载项目文件
1. 将本文件夹（包含index.html、script.js、tailwind.config.js和.gitignore文件）中的所有文件打包成ZIP文件
2. 或者直接复制这些文件到一个单独的文件夹中

### 第四步：上传文件到GitHub
1. 进入你刚刚创建的仓库页面
2. 点击 "Add file" 按钮，选择 "Upload files"
3. 拖拽或点击选择你的项目文件（index.html、script.js、tailwind.config.js和.gitignore）
4. 拉到页面底部，在 "Commit changes" 框中可以输入："上传智撰云网站文件"
5. 点击 "Commit changes" 按钮

### 第五步：开启GitHub Pages
1. 在仓库页面点击顶部的 "Settings" 选项卡
2. 在左侧菜单中找到并点击 "Pages"
3. 在 "Build and deployment" 部分，找到 "Source" 下拉菜单
4. 选择 "Deploy from a branch"
5. 在 "Branch" 下拉菜单中选择 "main" 分支，然后点击保存
6. 等待几分钟，GitHub会自动部署你的网站
7. 部署成功后，页面顶部会显示你的网站URL：`https://你的用户名.github.io`

### 第六步：访问你的网站
1. 复制显示的URL
2. 在浏览器中粘贴并访问
3. 你的智撰云网站现在已经成功部署到GitHub Pages上了！

## 注意事项
- GitHub Pages部署可能需要几分钟时间才能生效
- 如果网站不显示，请刷新浏览器或等待几分钟后再试
- 如果遇到问题，请检查文件名是否正确，特别是index.html必须使用这个名称
- 确保没有上传node_modules等不必要的文件夹

## 如何更新网站
如果以后需要更新网站内容：
1. 直接在GitHub仓库页面编辑文件
2. 或者重新上传修改后的文件
3. 提交更改后，GitHub Pages会自动更新网站
